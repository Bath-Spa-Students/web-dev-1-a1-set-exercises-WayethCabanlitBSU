HTML: HyperText Markup Language

  • HTML stands for HyperText Markup Language.
  • It's the standard markup language for creating web pages.
  • HTML consists of elements enclosed in angle brackets, which define the structure of a webpage.
  • Elements can include headings, paragraphs, links, images, lists, and more.
  • HTML documents are structured using tags like <html> , <head> , <body> etc.
  • Tags can have attributes that provide additional information about elements.
  • Browsers interpret HTML to render web content visually.

BASIC HTML TAGS

HEADINGS TAG

  • Headings are defined using <h1> to <h6> tags, where <h1> represents the highest level of heading (most important), and <h6> represents the lowest level.
  • <h1> is typically used for the main title of the page, while <h2> to <h6> are used for subheadings and nested content.
  • Search engines and screen readers use heading tags to understand the structure and importance of content.

PARAGRAPH TAG

The <p> tag in HTML defines paragraphs of text, providing a structured way to organize and display cohesive blocks of content on a webpage. It's fundamental for readability, accessibility, and styling through CSS.

LINE BREAK TAG

  • The <br> tag inserts a line break in the text, causing subsequent content to start on a new line.
  • It's a self-closing tag and does not require a closing tag.
  • Unlike paragraphs, which create a vertical space between blocks of text, <br> only creates a line break within the same block of text.
  • It's commonly used when you want to break a line of text without starting a new paragraph, such as in addresses, poetry, or song lyrics.

ANCHOR TAG

  • Hyperlinks created with anchor tags allow users to navigate between different web pages or to specific sections within the same page.
  • Anchor tags require the href attribute, which specifies the URL of the page the link goes to.
  • The text or content between the opening <a> and closing </a> tags is what users click on to follow the link, known as the anchor text.

DIV TAG

  • The <div> tag is used to define a division or a section in an HTML document. It's a block-level element, meaning it typically starts on a new line and stretches the full width of its parent container by default.
  • <div> elements are often used to group together and organize other HTML elements, such as text, images, forms, and other elements.
  • They provide a way to apply styling and layout to groups of content using CSS.

SPAN TAG

  • The <span> tag is used to mark a section of text or inline content for styling or scripting purposes.
  • It's an inline element, meaning it does not start on a new line and only takes up as much width as necessary.
  • <span> elements are commonly used to apply styles, such as color, font size, or font weight, to a specific portion of text within a larger block of text.

LIST TAGS

ORDERED LIST TAG

  • <ol>: Defines an ordered list.
  • <li>: Defines each item in the list.

UNORDERED LIST TAG

  • <ul>: Defines an unordered list.
  • <li>: Defines each item in the list.

CSS: Cascading Style Sheet

CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation and styling of HTML documents.

  • CSS defines how HTML elements are displayed on a webpage, including layout, colors, fonts, spacing, and more.
  • CSS separates the structure (HTML) of a webpage from its presentation (styling), allowing for better organization and maintainability of code.
  • CSS properties define the style aspects of HTML elements, such as color, font-size, margin, padding, etc. Each property has one or more values that
  • CSS selectors are used to target specific HTML elements to apply styles. Selectors can target elements by tag name, class, ID, attribute, or relationship with other elements.

INLINE CSS

    Inline CSS is applied directly to an individual HTML element using the style attribute. This method is useful for applying unique styles to specific elements but is generally not recommended for large-scale styling due to reduced maintainability and readability.

INTERNAL CSS

    Internal CSS is defined within the <style> tag in the <head> section of an HTML document. It applies styles to multiple elements within the same HTML document. Internal CSS is more organized than inline CSS but still affects only the current HTML file.

External CSS

    External CSS is defined in a separate .css file and linked to the HTML document using the <link> tag in the <head> section. This approach promotes clean separation of concerns, making it easier to maintain and apply consistent styles across multiple HTML files.
